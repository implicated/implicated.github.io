Standard File and Directory Set

Antora Configuration File

  • antora-playbook.yml

    • Set site start page.

    • Set site content.

    • Set site UI.

    • Set site out path.

site:
  title: My Demo Site
  url: https://docs.demo.com
  start_page: component-b::index.adoc (1)
content:
  sources:
  - url: https://gitlab.com/antora/demo/demo-component-a.git
  - url: https://gitlab.com/antora/demo/demo-component-b.git
    branches: [v2.0, v1.0]
    start_path: docs
ui:
  bundle:
    url: https://gitlab.com/antora/antora-ui-default/-/jobs/artifacts/HEAD/raw/build/ui-bundle.zip?job=bundle-stable
    snapshot: true
1 Site home page.

Module Configuration File

  • antora.yml

    • Required Key name and version.

    • Set modules nav.

name: implicated (1)
title: Implicated Blog
version: '1.0'
display_version: 'Bate Version'
start_page: ROOT:hello.adoc (2)
prerelease: -beta.1
nav:
- modules/ROOT/.nav.adoc
